1. Base64 Encoder + Decoder

Base64 is a group of similar binary-to-text encoding schemes that represent binary data in an ASCII string format by transforming it into a radix-64 representation.
The term Base64 originates from a specific MIME content transfer encoding.

2. A1Z26 Encoder + Decoder

A1Z26 is a simple substitution cipher that replaces each letter of the alphabet with its corresponding number.
For example, A becomes 1, B becomes 2, and so on until Z becomes 26.

You should need this in the very first levels.

3. Morse Code Encoder + Decoder

Morse code is a method used in telecommunication to encode text characters as standardized sequences of two different signal durations, called dots and dashes.
Fun fact: Samuel Morse developed the Morse code in the 1830s and 1840s, which is considerably a very long time ago.

4. Caesar Cipher Encoder + Decoder

The Caesar cipher is a type of substitution cipher in which each letter in the plaintext is shifted a certain number of places down or up the alphabet.
For example, with a shift of 1, A would be replaced by B, B would become C, and so on. The method is apparently named after Julius Caesar, who used it in his private correspondence.
